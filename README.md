# Pet Commerce Analytics

## Project Overview
반려동물 커머스 데이터 분석 및 고객 관리 자동화 프로젝트입니다.

## Dataset
- 고객 데이터
- 주문 데이터
- 상품 데이터
- 사용자 설정 데이터

## Key Features
- 고객 세분화
- 매출/주문 분석
- Streamlit 대시보드
- 사용자별 관리 기능

## Project Structure
- `app.py`: Streamlit 메인 실행 파일
- `src/`: 분석 및 기능 모듈
- `models/`: 모델/로직 파일
- `assets/`: 이미지 및 기타 리소스
- `users.yaml`: 사용자 설정 정보

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
